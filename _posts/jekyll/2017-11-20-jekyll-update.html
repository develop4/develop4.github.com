Updating Jekyll to use bundler was a bit more complex than anticipated.

The first item on the agenda was to make sure that my Ruby environment and all the Gems were upto date.

  • gem update --system
  • gem install rails
  • gem update

The next task was to create the Gemfile required by the bundler. 

source "https://rubygems.org"
#     bundle exec jekyll serve
# This is the default theme for new Jekyll sites. You may change this to anything you like.
gem "minima", "~> 2.0"
gem "github-pages", group: :jekyll_plugins
group :jekyll_plugins do
  gem "jekyll-feed", "~> 0.6"
end
gem 'tzinfo-data', platforms: [:mingw, :mswin, :x64_mingw, :jruby]
